# apache 2 ssl

Compiles apache 2 in an opinionated way and generates a debian package.  On install the debian package generates a self-signed certificate for https.

## Info
Code in src directory is a direct download of apache source code:

Current version is 2.0.63

http://httpd.apache.org/download.cgi

## Warning

This is woefully obsolete and just present as a historical archive.  Odds are it won't build on any version of debian newer than 2010 or so.
